# poply-ui

Beautiful, ready-made popup components. 150+ themes. Zero friction.

## Installation

```bash
npm install poply-ui
```

## Quick Start

```javascript
import Poply from 'poply-ui';
import 'poply-ui/dist/poply-ui.css';

Poply.alert({ 
  theme: 'ios-glass', 
  title: 'Hello! 👋', 
  message: 'Welcome to poply-ui.' 
});
```

See the [official documentation](https://poply-ui.dev) for more details.
